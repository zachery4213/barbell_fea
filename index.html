Olympic Barbell Under Various Plate Loading

I modelled a barbell using Olympic-standard dimensions and used finite element analysis to compare how increasing plate loads changed stress, deformation, and safety factor. The setup uses a centre shoulder-width support region with remote loading points at both sleeve ends.

01 / Simulation Setup

The analysis was organized around three plate-loading scenarios. Each case used the same geometry and constraint strategy so the differences in response came from the increased loading.

A

Geometry Intent

The barbell was treated as the core load-carrying body. Sleeve loading regions, the central shoulder support, and the bar span were separated conceptually so the FEA results could be read like a bending study.

B

Squat Boundary Condition

A remote displacement was placed across the centre of the bar at approximately shoulder width. This simplified the lifter contact while keeping the support in the physically meaningful region.

C

Plate Loading

Remote regions on both sides represented the plate stacks. The cases increased from one 25 lb plate per side to two 45 lb plates per side.
